How Color Converter Works

This converter allows you to input a color in one of the following formats:

HEX: A six-digit hexadecimal code (e.g., #FF5733).
RGB: Red, Green, and Blue values (e.g., rgb(255, 87, 51)).
HSL: Hue, Saturation, and Lightness values (e.g., hsl(14, 100%, 60%)).

Once you provide one of these inputs, the tool instantly converts it into the other two formats, giving you a complete view of the color in all three representations.
